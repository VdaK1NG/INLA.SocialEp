# Package index

## INLA Model Wrappers

Wrapper functions for several advanced disease mapping models in INLA.

- [`inla.SpANOVA.2x2()`](https://vdak1ng.github.io/INLA.SocialEp/reference/inla.SpANOVA.2x2.md)
  : Apply SpANOVA modelization using a wrapper function in INLA

## Model evaluation

Evaluate and interpret models.

- [`inla.MARB()`](https://vdak1ng.github.io/INLA.SocialEp/reference/inla.MARB.md)
  : Calculate MARB between fitted values in INLA and simulated values
- [`inla.MedARB()`](https://vdak1ng.github.io/INLA.SocialEp/reference/inla.MedARB.md)
  : Calculate MedARB between fitted values in INLA and simulated values
- [`inla.MRRMSE()`](https://vdak1ng.github.io/INLA.SocialEp/reference/inla.MRRMSE.md)
  : Calculate MRRMSE between fitted values in INLA and simulated values
- [`inla.MedRRMSE()`](https://vdak1ng.github.io/INLA.SocialEp/reference/inla.MedRRMSE.md)
  : Calculate MedRRMSE between fitted values in INLA and simulated
  values
- [`inla.null.sp()`](https://vdak1ng.github.io/INLA.SocialEp/reference/inla.null.sp.md)
  : Extract null effects from SR-ANOVA models

## Accessory Functions

Prepare datasets for analysis.

- [`inla.full.graph()`](https://vdak1ng.github.io/INLA.SocialEp/reference/inla.full.graph.md)
  : Make a fully connected graph for INLA

  : Mini OTU abundance dataset - preprocessed

  : Mini OTU abundance dataset with 3 categorical variables
- [`sim_sp_ef`](https://vdak1ng.github.io/INLA.SocialEp/reference/sim_sp_ef.md)
  : Mini OTU abundance dataset
